Download Page for "adb" tool.

If you are not using Android operating system 4.2.2 or newer, you can ignore this message.

If you are upgrading to the latest version of the Android operating system, version 4.2.2, you will need to also upgrade a program on your computer, or else you won’t be able to connect to the phone with the USB cable. Look here for some background.

If you are using App Inventor with the USB cable, and you are upgrading the phone to Android version 4.2.2 you should:

  1. Find the folder where the Appinventor Setup software was installed, and locate the "adb" program (also called "adb.exe" on Windows)
  2. Download the new version of the "adb" program appropriate to your operating system.
    • adb-MacOSX.zip
    • adb-Windows.zip
    • adb-Ubuntu.zip
  3. Extract and replace the adb program with this new version

The next time you connect your phone to your computer with USB debugging turned on, you’ll be asked to "Allow USB Debugging?", say "OK". App Inventor should now work as before. We are planning to release a new version of the App Inventor setup software in a few weeks that will make this extra step unnecessary.
