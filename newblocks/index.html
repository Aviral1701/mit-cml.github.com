Welcome to App Inventor "newblocks" pilot

Welcome to the pre-Alpha test of the next generation of App Inventor. The most significant change in this version is the moving of the blocks editor into the browser instead of residing in a separate Java program.

A lot about this new version is very very cool. A lot about this version is very very unstable! Even the exact name has not been fully decided. Colloquially we have referred to this version as "newblocks."

Keep in mind that this is very much a work in progress. We believe this version works well enough to give you a feel for where we are going, and we welcome feedback. However it is NOT SUITABLE FOR PRODUCTION OR CLASS USE.

This version is also undergoing continuous update and improvement. Do not be surprised if it looks different from day to day. We know that not all of it works (though do let us know when you find something broken). We may also break it (accidentally) from time to time.

Access to this version is by invitation only. Chances are if you are reading this note, you have been invited and have been added to our list of allowed users. But keep in mind that not everyone can use this version (yet).

Finally. Thank You for volunteering to help us improve App Inventor! We very much appreciate your help and feedback!

Only wireless device connection is supported: You will need the new MIT NewBlocks Companion App

You will need to download the MITNewBlocksCompanion.apk to do incremental development. This app is not available in the Google Play Store. You will need to download it here and “side load” it into your device (if you need instructions on how to do so, please contact us). We do not yet have an emulator that works with this version, but we are planning on having one. Do not worry if you already have the MIT AICompanion app on your device because this newer version can co-exist with the version that you already have.

Testing Instructions

For this testing phase we are primarily interested in whether the blocks language have any problems, for example can you create the same application in the current version of AppInventor with this new version.

You will definitely notice many changes in this new version for example, we now have “Local Variables”, and missing functionality such as “Type Blocking”. Please remember that this is still a work-in-progress and you will see new functionality appear.

Please start by:

  1. Recode applications that you have created, acquired or have used in your classes and see if they still work. Also, if you could send us a zip file of your application that would be helpful too. If you can’t find any applications there is a collection here.
  2. Code your apps using local variables.
  3. Create application that use multiple screens.
