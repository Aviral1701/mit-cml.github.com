The system at this web site is App Inventor Classic. App Inventor Classic is obsolete.

App Inventor Classic IS SHUTTING DOWN on July 15, 2015. Important Information, Please Read

If you are creating new apps, or running classes with App Inventor, you should switch to App Inventor 2, which is available at http://ai2.appinventor.mit.edu.

Projects downloaded from App Inventor 1 cannot be loaded in App Inventor 2. MIT has provided a converter tool to help transfer projects from App Inventor 1 to App Inventor 2. This converter is available at http://convert.appinventor.mit.edu

The converter won't produce a complete automatic translation: You'll probably need to do some hand modification to the result of the conversion.
