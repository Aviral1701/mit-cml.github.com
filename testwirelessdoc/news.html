Wireless Phone Connection Test -- News November 15, 2012

The Testwireless App Inventor server has been updated. It is now up-to-date with the current beta release (v130a). The most noticeable change you will see on the wireless side will be that the "Connect to Phone" button is gone from the blocks editor. You don't need it! Just enter your code on the phone (or scan the QR Code) and the blocks editor will pick up on it.

We are getting very close to releasing this code on our normal (beta.appinventor.mit.edu) server. In the next few days I will be putting up a short survey that I would appreciate it if you could fill out. I'm interested in knowing what phones you have used the wireless code on and how well it worked (or didn't!). I'm also interested in general feedback. Thanks for your help!

Note: This version requires the newest Companion App (version 1.10L).

We have released a new version of the MIT App Inventor Debug App. This version no longer expires. However it is tied to the version of the MIT App Inventor Server it was built for. Wireless Phone Connection Test -- NEWS September 6, 2012

This version changes the way you connect to your phone. The "Connect Wireless" button is gone. Instead under the connect menu there is now a "WiFi" option which does the same thing. Plus, if you exit the Debugger App you can start a new one and connect to the same blocks editor (previously you had to quit and restart the blocks editor). Just start a new debugger app on your phone. The newest version has a "Restart" button. Select "Restart" on your phone and then reselect "WiFi" from the blocks editor connect menu. It should reconnect without you having to enter a new code or scan a barcode.

This version of App Inventor has new features that will be released on the main production server soon. In the meantime you may have problems downloading projects from here and then uploading them to beta.appinventor.mit.edu because your blocks will be too new! This problem should resolve itself in a few days when release v129 is out.
