Alert to our Macintosh users!
Apple on January 31, 2013 made a change to the Macintosh operating system blocking
applications using Java 6 from running (Apple Blocks Java), this includes App Inventor.
We were taken by surprise by Apple's decision, but there is a solution and we are working
to post it tomorrow. We are sorry for any inconvenience this has caused our users.
