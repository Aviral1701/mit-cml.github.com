App Inventor Source

Welcome to the App Inventor Source Code Repository. For more information on App Inventor go to Main App Inventor Website.

We have moved the MIT App Inventor source code from the Mercurial version control system to git. The location has also moved from code.google.com to github.com.

You can find a lot of information on GitHub and the git system itself at github.com.

Feel free to peruse the source. This is the actual source code we use to build the MIT App Inventor site. You are welcome to use it to build your own version. However we cannot provide support for this at the current time.
